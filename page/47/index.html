Debian远程SSH汉字utf-8乱码解决

aptitude install locales
dpkg-reconfigure locales

配置编码进入选择(默认也选择)：

en_US.UTF8

查看默认字符集是否是en_US.UTF-8

$ vim /etc/default/locale
LANG=en_US.UTF-8

配置编码中文utf-8

dpkg-reconfigure locales

系统环境默认的区域设置成

zh_CN.UTF-8

退出重新登陆ssh 才起作用
